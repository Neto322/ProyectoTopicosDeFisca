Cada propiedad del Shader tiene su referencia ya hecha. Aquí está la lista:

Shader Disco
-Color = _Disco_Color
	Podría ser que el usuario elija el color del Disco (?)
-RimPower = _Disco_RimPower
	Este cuando lo decrementas, el disco se vuelve más "brilloso" o iluminado.
	Este valor tiene que subir con cada golpe.

Shader Holograma
-BaseColor = _Holograma_Color
	Solo hay dos colores, que sería el rojo y azul para los equipos. Hay dos materiales que tienen este Shader:
		PorteriaAzul y PorteriaRoja
-Velocidad del Holograma = _Holograma_Velocidad
	Puede incrementar con el tiempo el valor de este, o nomás detenerse cuando le metan gol.
-EmissionPower = _Holograma_EmissionPower
	Si incrementa el valor se vuelve más brilloso, usalo como quieras.

ShaderMetalico
-Smoothness = _Metalico_Smoothness
	Este valor en el material del Piso puede bajar para que no se vea tan reflejante después de un rato, como que se desgasta la cancha. O puede quedarse igual.

ShaderNeon
-BaseColor = _NeonBaseColor
	Es el color base de las luces Neon del estadio nomás. Puedes cambiarlo en cuestión del equipo ganador.
-RimPower = _NeonRimPower
	Más brillo o iluminación, manejalo como quieras
-RimColor = _NeonRimColor
	Es el color del brillo, puede cambiarse igual que el BaseColor

ShaderPlastico
-RimPower = _Sombrerito_RimPower
	Es el brillo que tiene el sombrerito, cuanto mayor golpes tenga que se ilumine más. Cada decremento es más brillo.

Las demás propiedades no es necesario un cambio de color o Shader.
